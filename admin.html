📖 Cara Update Galeri
  1. Upload foto baru ke folder foto/ di repository GitHub
  2. Di admin ini, tambah momen baru dengan URL foto/nama-file.jpg
  3. Klik Ekspor JSON → Download photos.json
  4. Di GitHub, buka file photos.json → klik ✏️ Edit → paste isi baru
  5. Klik Commit changes — galeri terupdate dalam 30 detik!
